# PáginaFácil Premium — versão pronta para publicar

Plataforma Next.js com Firebase e checkout Mercado Pago.

## Iniciar no Windows

1. Clique em `CONFIGURAR-PAGAMENTO.bat` e informe seu Access Token.
2. Clique em `INSTALAR-E-INICIAR.bat`.
3. Abra `http://localhost:3000`.

## Publicar

Leia `GUIA-PARA-PUBLICAR.txt`.

## Pagamento

O checkout utiliza o Mercado Pago Checkout Pro. O Access Token permanece somente no servidor, dentro de `.env.local` ou nas variáveis de ambiente da hospedagem.

## Teste técnico

Projeto compilado com sucesso usando `npm run build` em Next.js 16.2.10.
