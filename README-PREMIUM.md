# PáginaFácil Premium 6.0

Plataforma em Next.js 16 com:

- Landing page premium e responsiva
- Dashboard profissional
- Criador de sites com IA (interface e fluxo existentes)
- Logo Studio
- Banner Studio
- Planos Essencial, Profissional e Agência
- Templates, analytics, leads, domínios e configurações
- Firebase Authentication e Firestore
- CSS puro, sem dependência do Tailwind

## Como iniciar no Windows

1. Extraia o ZIP em uma pasta nova.
2. Clique duas vezes em `INSTALAR-E-INICIAR.bat`.
3. Acesse `http://localhost:3000`.

## Pelo terminal

```bash
npm install
npm run dev
```

## Observação

As telas de Logo Studio e Banner Studio já estão prontas e interativas no front-end. Para gerar imagens reais por IA, será necessário conectar uma API de geração de imagens no servidor. Os planos são telas comerciais; pagamentos recorrentes precisam ser conectados ao Mercado Pago, Asaas ou Stripe.
